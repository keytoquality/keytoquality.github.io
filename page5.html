K2Q FRACAS Module

Module Description

FRACAS is an acronym for Failure Reporting and Corrective Action System. This type of system is designed to record and classify problems related to a product or process and their associate root causes so that improvement activities can be carried out.

There is no single standard on how to implement a FRACAS system. There are many software packages out there today that cater for this, but they are expensive, cumbersome, not intuitive and in many cases give you so much freedom that it is not clear if the implementation is correct without their expensive consultation.

The K2Q FRACAS module uses a simple yet versatile incident reporting form, allowing you to confidently collect problem details about any type of incident, and then correct those problems through a consistent, structured problem resolution process. The FRACAS module comes pre-built with ready to use 8D and DMAIC workflows.
